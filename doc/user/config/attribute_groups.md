# Mints::User::Config::Appointments

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_attribute_groups_data_types #=> Get data types used in attribute groups.

user.get_attribute_groups(options) #=> Get a collection of attribute groups.

user.get_attribute_group(id) #=> Get a collection of attribute groups.

user.create_attribute_group(data) #=> Create an attribute group with data.

user.update_attribute_group(id, data) #=> Update an attribute.
```
