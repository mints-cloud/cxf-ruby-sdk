# Mints::User::Config::Calendar

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_calendars(options) #=> Return a collection of calendars.

user.get_calendar(id, options) #=> Return a single calendars.

user.create_calendar(data) #=> Create a calendar.

user.get_calendars(options) #=> Return a collection of calendars.

user.update_calendar(options) #=> Return a collection of calendars.

user.delete_calendar(id) #=> Delete a calendar.
```
