# Mints::User::Content

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_public_images_url #=> Get public images url.

user.get_keywords(options) #=> Get a collection of keywords.

user.get_keyword(id) #=> Get a keyword.

user.create_keyword(data) #=> Create a keyword.

user.update_keyword(id, data) #=> Update a keyword info.

user.get_stages(options) #=> Get a collection of stages.

user.get_stage(id) #=> Get a stage.

user.create_stage(data) #=> Create a stage.

user.update_stage(id, data) #=> Update a stage info.

```
