# Mints::User::Content::Pages

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_page_groups #=> Get content page groups.

user.get_pages(options) #=> Get a collection of content pages.

user.get_bundles(options) #=> Get a collection of content pages.

user.get_page(id) #=> Get a content page.

user.create_page(data) #=> Create a content page.

user.update_page(id, data) #=> Update a content page info.

user.delete_page(id) #=> Delete a content page
```
