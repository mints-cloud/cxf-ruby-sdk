# Mints::User::Helpers::ObjectFolders

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_object_folders(options) #=> Get a collection of object folders.

user.get_object_folder(id, options) #=> Get an object folder info.

user.create_object_folder(data) #=> Create an object folder with data.

user.update_object_folder(id, data) #=> Update an object folder info.

user.delete_object_folder(id) #=> Delete an object folder.
```
