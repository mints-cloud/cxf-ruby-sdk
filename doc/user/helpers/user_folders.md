# Mints::User::Helpers::UserFolders

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_user_folders(options) #=> Get a collection of user folders.

user.get_user_folder(id, options) #=> Get an user folder info.

user.create_user_folder(data) #=> Create an user folder.

user.update_user_folder(id, data) #=> Update an user folder info.

user.delete_user_folder(id) #=> Delete an user folder.
```
