# Mints::User::Ecommerce::VariantOptions

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_variant_options(options) #=> Get a collection of variant options.

user.get_variant_option(id, options) #=> Get a variant options info.

user.create_variant_option(data) #=> Create a variant option.

user.update_variant_option(id, data) #=> Update a variant option info.
```
