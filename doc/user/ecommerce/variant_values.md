# Mints::User::Ecommerce::VariantValues

```ruby
user = Mints::User.new(mints_url, api_key, session_token)

user.get_variant_values(options) #=> Get a collection of variant values.

user.get_variant_value(id, options) #=> Get a variant value info.

user.create_variant_value(data) #=> Create a variant value.

user.update_variant_value(id, data) #=> Update a variant value info.
```
